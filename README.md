# note

# 功能需求：
## 实现一个记录笔记的小程序，能够增删查改


# 前端: wepy框架
## 快速开始：
	进入到  wx，执行：cnpm install
	开启： npm run dev




# 后台：node + koa + mongole
## 快速开始：
	进入到  service，执行：cnpm install
	安装数据库mongodb，官网 ： https://www.mongodb.com/download-center
	启动数据库mongodb： 进入MongoDB\bin目录，执行：mongod --dbpath D:\mongodb\data\db
	开启： node app.js


# 注意：
## 在本地前后端开发时，要同时开启mongodb数据库服务
